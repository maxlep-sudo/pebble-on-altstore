# Pebble-on-AltStore

Pebble-on-AltStore is an open-source companion app for Pebble smartwatches, designed for iOS devices and intended for sideloading via AltStore. This project aims to provide Pebble users with a simple way to regain watch connectivity, notifications, and watch management on modern iPhonesâ€”even after the official Pebble app has been removed from the App Store.

## Features

- Pair and manage Pebble watches on iOS
- Send notifications and alerts to Pebble
- Install watchfaces and apps from the phone
- Sync health and activity data (planned)
- Open-source and privacy-respecting
- Easy sideloading via AltStore

## Installation

This app is designed to be sideloaded using [AltStore](https://altstore.io):

1. Clone or download this repository.
2. Open the project in Xcode and build the `.ipa` file.
3. Use AltStore on your iOS device to sideload the app.
4. Connect your Pebble using Bluetooth and start using the app!

See [docs/INSTALL.md](docs/INSTALL.md) for step-by-step details (coming soon).
